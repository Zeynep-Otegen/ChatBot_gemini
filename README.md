# ChatBot
ChatBot (Gemini API – WinForms)

Bu proje, Google Gemini API kullanılarak geliştirilmiş basit bir Windows Forms (C#) masaüstü sohbet uygulamasıdır.
Kullanıcıdan alınan metin, Gemini dil modeline gönderilir ve modelin ürettiği yanıt arayüzde gösterilir.

# Özellikler

Google Gemini API entegrasyonu

Windows Forms (WinForms) kullanıcı arayüzü

Asenkron HTTP istekleri (HttpClient)

JSON tabanlı veri alışverişi (Newtonsoft.Json)

API anahtarını koda gömmeden, harici dosyadan okuma

# Kullanılan Teknolojiler

C# (.NET / WinForms)

Google Gemini API

Newtonsoft.Json

HttpClient

# Kullanım

Uygulamayı başlat

Mesaj kutusuna metni gir

Gönder butonuna bas veya Enter tuşuna bas

Gemini modelinin yanıtını sohbet alanında gör

# Kullanılan Model

gemini-2.5-flash
